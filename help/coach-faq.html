    How to add a new PBC Cycle

    1.    Tap Add New PBC Cycle bar on the home page.

    2.    Enter the requested information in the form: PBC Cycle number, PBC Cycle name, term, and the PBC Cycle begin/end dates.

    3.    Click Create.

     

    How to edit an existing PBC Cycle’s information

    1.    Click the correct PBC Cycle on the home page.

    2.    Go to the Assignments tab.





    3.    Click Edit Content in the upper right.

    4.    Tap Edit This PBC Cycle bar.

    5.    Change the appropriate information.

    6.    Click Update.
