GENERAL NAVIGATION FUNCTIONS

  • As Cycle Steps are created, they will appear on each Cycle’s main menu as Dialogue Bars.
  • Dialogue Bars serve as toggle switches to open and close each Dialogue Bar menu.
  • Return to Coaching Companion Homepage at any point by Selecting PBC CYCLES on the menu at the top right of each page.
  • Edit PBC Cycles by selecting a PBC Cycle, then select the Edit PBC Cycle button to the right of the of the PBC Cycle name at the top of the page and an Edit this PBC Cycle bar will appear directly below. Select the Edit this PBC Cycle bar to begin editing.

Return to Cycle Steps menu at any point in the process.

  1. PBC Cycles:
    1. Create PBC Cycle
      1. Begin on the Coaching Companion Homepage.
      2. Select the Add New PBC Cycle bar.
      3. Enter a PBC Cycle Name in the PBC Cycle Name window.
      4. Select the PBC Cycle Begins window and select a start date from the drop-down calendar.
      5. Select the PBC Cycle Review Date window and select a date from the drop-down calendar.
      6. Select the Create button, and you will return to the Coaching Companion Homepage.
      7. Select the newly created PBC cycle that appears on the Homepage.
      8. Selected PBC Cycle will open.
      9. Select Add PBC Cycle Steps bar.
    2. Add Action Plan Goal
      1. Select the Action Plan Goal bar.
      2. Enter goal for this coaching cycle in the What is your goal for this coaching cycle? window.
      3. Enter how you know when you have achieved this goal in the How will you know when you have achieved this goal? window.
      4. Select the Create button.
    3. Add PBC Cycle Steps
      1. Select the Add PBC Cycle Steps bar.
      2. Dialogue bars will now appear above the Add PBC Cycle Steps bar.
      3. Select the Add an Action Plan Step bar.
      4. Enter Action Plan Step Name in the Action Plan Step window.
      5. Select the Action Plan Start Date window and select a date from the drop-down calendar.
      6. Select the Action Plan End Date window and select a date from the drop-down calendar.
      7. Select the Action Plan Step Assignee bar.
      8. Select an Assignee’s name and drag-and-drop it into the Select Action Plan Step Assignee window.
      9. Assignee’s name will appear in Selection Action Plan Step Assignee window.
      10. Select the Comments window to add comments.
      11. Select the Arrow Toggle in the Progress window and select progress status from the drop-down menu that appears.
    4. Add Media Resources
      1. To add media resources to this Action Plan you must first Add an Action Plan Step.
      2. Select the Add an Action Plan Step bar.
      3. Select the Create button.
      4. Select the Associated Resources bar.
      5. Select the Add Resource from Resource Library button.
      6. Resource Library Homepage will open
      7. Select Default, Video Glossary, or Teaching Domains to access each category’s Drop-down Menu.
      8. Select the View button to view a resource.
      9. To choose a video to be associated with your Action Plan Step select the Associate This Resource to bar at the bottom of the viewing page.
      10. To continue browsing Associated Resources select the Back to Previous Page bar at the bottom of the page.
    5. Add Your Own Media
      1. Select the Add New Media bar.
      2. Select the Choose File button to upload or select Select from Your Current Videos bar and then select Submit Selected Video button.
      3. Select create button.
    6. Add Focused Observation
      1. Return to the PBC Cycle homepage.
      2. Select a PBC Cycle.
      3. Select Add PBC Cycle Steps bar.
      4. Select on Add a Focused Observation bar.
      5. Select on the arrow toggle in the What Action Plan Step are you Observing? window.
      6. Select an Action Plan Step from the drop-down menu that appears.
      7. Select the Observers bar.
      8. Select an Observer Name and drag-and-drop it into the Selected Observers window.
      9. Select the Observed Participants.
      10. Select on Observed Participants Name and drag-and-drop it into the Selected Observed Participants window.
      11. Select on the Focus window and enter the Focus of the Observation.
      12. Select on the Date window and select a date from the drop-down calendar.
      13. Select on the Preparation Notes window and enter preparation notes.
      14. Select the Create button.
    7. View Reflection and Feedback Responses
      1. Return to the PBC Cycle homepage.
      2. Select a PBC Cycle.
      3. Select on View Reflection and Feedback Responses bar.
      4. Select on View button.
      5. Select on Associated Resources or Add Comments bar to toggle open and close, and follow prompts.
    8. Review to Complete or Add an Action Plan Step to the Coaching Cycle
      1. Return to the PBC Cycle homepage.
      2. Select a PBC Cycle.
      3. Select on the Review bar and follow prompts to Complete or Add an Action Plan Step in the cycle.
    9. Return to Cycle Steps menu at any point in the process.
      1. Return to the PBC Cycle homepage.
      2. Select a PBC Cycle.
      3. Select on PBC Cycle Name at top left of page
